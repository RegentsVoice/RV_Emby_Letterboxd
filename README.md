# Emby Letterboxd

Adds a "Letterboxd" button to the external links section (IMDb, TMDB, etc.) on Emby media server pages. The button links directly to the corresponding film page on Letterboxd using the IMDb ID.

## Features

- Automatically detects the IMDb link on any Emby item page.
- Inserts a Letterboxd button next to the existing external links.
- Works with dynamic page updates (no need to reload the page when navigating between items).
- Supports any Emby server address (not hardcoded to a specific IP/port).

## Installation

1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/).
2. Click the link: [emby-letterboxd.user.js](https://github.com/RegentsVoice/RV_Emby_Letterboxd/raw/refs/heads/main/emby-letterboxd.user.js)
3. The userscript manager should prompt you to install it. Confirm.

## Usage

Once installed, simply browse your Emby library. When you open a movie page, a "Letterboxd" link will appear among the external links (e.g., IMDb, TMDB). Click it to open the film's Letterboxd page.

## Compatibility

- Tested with Emby Server 4.8+.
- Works in any modern browser with a userscript manager.

# Emby Letterboxd

Добавляет кнопку "Letterboxd" в блок внешних ссылок (IMDb, TMDB и др.) на страницах медиа-сервера Emby. Кнопка ведёт на соответствующую страницу фильма на Letterboxd, используя IMDb ID.

## Возможности

- Автоматически определяет ссылку IMDb на любой странице элемента Emby.
- Вставляет кнопку Letterboxd рядом с существующими внешними ссылками.
- Работает с динамическими обновлениями страницы (не требует перезагрузки при переходе между элементами).
- Поддерживает любой адрес сервера Emby (не привязан к конкретному IP/порту).

## Установка

1. Установите менеджер пользовательских скриптов, например [Tampermonkey](https://www.tampermonkey.net/) или [Violentmonkey](https://violentmonkey.github.io/).
2. Нажмите на ссылку: [emby-letterboxd.user.js](https://github.com/RegentsVoice/RV_Emby_Letterboxd/raw/refs/heads/main/emby-letterboxd.user.js)
3. Менеджер скриптов предложит установить его. Подтвердите установку.

## Использование

После установки просто просматривайте вашу медиатеку Emby. При открытии страницы фильма среди внешних ссылок (например, IMDb, TMDB) появится ссылка "Letterboxd". Нажмите на неё, чтобы открыть страницу фильма на Letterboxd.

## Совместимость

- Протестировано с Emby Server 4.8+.
- Работает в любых современных браузерах с менеджером пользовательских скриптов.
